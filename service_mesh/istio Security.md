# istio Security

## 概览

Istio Security的目标是提高微服务及其通信的安全性，而不需要更改服务代码。 它负责：

为每个服务提供一个强大的身份，以表示其在集群和云之间的互操作性

保障服务间通信和终端用户与服务通信

提供密钥管理系统来自动化密钥和证书的生成，分发，轮换和撤销
